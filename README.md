# Telegram CRM Bot for Railway

Готовий бот для Railway. Працює через long polling, тому окремий домен і webhook не потрібні.

## Що вміє
- Один спільний бот, але окремі кабінети (workspace)
- Додавання клієнтів
- Пошук по телефону / ніку
- Рейтинг 0-5
- Окрема кнопка `⚠️ Низький рейтинг`
- Окрема кнопка `🌐 Глобальний список 0-2` з мінімальними даними
- Бонусні бали `ББ 🎁`
- Додавання працівників у кабінет

## Важливо для Railway
Якщо ви хочете, щоб база не зникала після redeploy, треба додати **Volume** і змонтувати його в `/data`.
Railway прямо вказує, що volumes потрібні для persistent data. Див. docs Railway: volumes reference.

## Швидкий запуск
1. Завантажте всі файли у GitHub репозиторій.
2. У Railway підключіть цей repo до сервісу.
3. У Variables додайте:
   - `BOT_TOKEN`
   - `DB_PATH=/data/crm.sqlite`
4. У сервісі додайте Volume і mount path `/data`.
5. Deploy / Redeploy.
6. В Telegram відкрийте бота і натисніть `/start`.

## Перше використання
- Власник бота натискає `🏢 Створити кабінет`
- Вводить назву системи
- Далі може додавати працівників і клієнтів

## Ролі
- owner - створює кабінет, додає працівників, бачить все в межах свого кабінету
- staff - може шукати та додавати клієнтів у межах свого кабінету

## Формат глобального низького рейтингу
Показується лише:
- ПІБ
- телефон або соцконтакт
- рейтинг
- назва кабінету
